One-Shot Learning of Manipulation Skills with Online Dynamics Adaptation and Neural Network Priors

NIPS 2015 Deep Reinforcement Learning Workshop , Under review for conference publication
  • Justin Fu,
  • Sergey Levine,
  • Pieter Abbeel
One of the key challenges in applying reinforcement learning to complex robotic control tasks is the need to gather large amounts of experience in order to find an effective policy for the task at hand. Model-based reinforcement learning can achieve good sample efficiency, but requires the ability to learn a model of the dynamics that is good enough to learn an effective policy. In this work, we develop a model-based reinforcement learning algorithm that combines prior knowledge from previous tasks with online adaptation of the dynamics model. We encode the prior experience into a neural network dynamics model, and adapt it online by progressively refitting a local linear model of the dynamics.

Robust Object Pose Tracking for Robotics

CS280 (Computer Vision)
  • Chelsea Finn,
  • Justin Fu,
  • Nopphon Siranart
We propose a neural network based approach for visual 3D object tracking, a task important for applications such as robotic manipulation and video scene understanding. We experiment with different architectures (recurrent and feedforward) and artificial data augmentation (occlusions, background, and lighting variation) to improve prediction accuracy. We also experiment with batch normalization, a technique used to both regularize and speed up neural network training.

Gesture Recognition from Accelerometer Data with Recurrent Neural Networks

VS265 (Neural Computation)
  • Justin Fu,
  • Siddartho Bhattacharya
We apply recently popular Long Short-Term Memory (LSTM) neural network architecture to the problem of gesture recognition from accelerometer readings. Using approximately 5 minutes of manually collected data, we were able to achieve 80% accuracy across 6 different gestures.

Baxter drawing

EE125 (Introduction to Robotics)
  • Justin Fu,
  • Hong Joo Kim,
  • Sara Seacat
We developed an algorithm for drawing images on paper given only a raw image as input. Our algorithm processes the image into strokes, and translates them into trajectories for a robot to execute. We were able to draw several cartoon pictures, although the noisy actuators and cameras on the Baxter robot we used somewhat limited the accuracy of the realizations of the images.
